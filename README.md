#  XJTLU_USA_CS_Application

美研收到边缘政治影响，项目难度忽高忽低，且各个项目之间内部还有改变，本帖只帮助各位梳理思路。为什么需要这个repo? 很多学生由于高考上来，第一次填申请，甚至大学才开始翻墙，对申请本就困惑，被中介赚走许多信息差。英国2-3w 美国 4-5w甚至有项目数量限制。在选校上Opencsapp有些过时加上这几年出来许多新项目导致。
